# Senior Data Architect - Digital Health Transformation Program

## Hiring Context
A Health Department within Government is recruiting for a Senior Data Architect role within a Digital Health Transformation Program.

## Contract Details
- Rolling contract (ASAP start)
- Full remote flexibility
- Day rate negotiable

## Role Purpose
Lead enterprise data architecture design across clinical, financial, and operational domains, ensuring health data is standardised, interoperable, secure, and accessible statewide.

## Responsibilities
- Define scalable enterprise data models.
- Establish governance frameworks and metadata standards.
- Define migration patterns and enterprise data blueprints.
- Lead architecture decisions across cloud and on-prem platforms.

## Key Experience Sought
- Designing enterprise-scale data architectures.
- Strong healthcare data modelling.
- Large-scale data warehouse/lake design, including medallion architecture and star/snowflake schemas.
- Microsoft Azure and Microsoft Fabric expertise.
- Data governance, metadata management, and privacy compliance.
- Secure data exchange protocols, including OAuth2, SAML, and TLS.

# Data Engineer - Job Description

## Role Overview
As a Data Engineer based in Melbourne, you will design scalable data lakes using Azure technologies, develop sophisticated ETL/ELT workflows for diverse datasets, and collaborate with multidisciplinary teams to deliver actionable insights. You will play a key role in embedding rigorous governance frameworks while ensuring optimal performance of all data assets.

## Key Responsibilities
- Develop, optimise, and maintain complex data pipelines.
- Build scalable ETL/ELT processes that enable seamless movement of data from diverse sources into centralised repositories for analytics and reporting purposes.
- Ensure comprehensive data quality, governance, and security measures are embedded throughout all stages of the data lifecycle, adhering to best practices and compliance standards.

## What You Bring
To excel as a Data Engineer in this role, your background should include deep familiarity with Microsoft's suite of analytics tools, particularly those focused on storage, orchestration, transformation, and visualisation. You will have demonstrated success managing complex projects involving multiple stakeholders while upholding rigorous standards for quality assurance.

- Extensive experience in data engineering or related fields with a proven track record of delivering enterprise-grade solutions in complex environments.
- Deep expertise in Microsoft Azure Synapse Analytics, Azure Data Lake Storage, and SQL Server for building end-to-end data platforms.
- Advanced proficiency in Python as well as strong command of SQL for querying large datasets efficiently.
- Demonstrated ability implementing robust data governance frameworks using tools like Purview while maintaining high standards for security and compliance.
